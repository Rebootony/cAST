cAST
====

C Preprocessor and Parser

[![Build Status](https://secure.travis-ci.org/scottfrazer/cAST.png)](http://travis-ci.org/scottfrazer/cAST)

Example
-------

```bash
$ cat source.c
```


```c
int main( int argc, char * argv[] )
{
  int y;
  y = factorial(6);
  printf("6! = %d\n", y);
}

int factorial(int x)
{
  if ( x == 1 )
    return 1;
  return x * factorial(x-1);
}
```


```bash
$ cast ast source.c
```
